通过 Git 贡献来添加你的团队成员卡片

如何贡献

  1. Fork 这个仓库
  2. 克隆到本地：git clone [你的仓库地址]
  3. 创建新分支：git checkout -b add-member-[你的名字]
  4. 在 index.html 中添加你的团队成员卡片
  5. 提交更改：git commit -m "添加团队成员: [你的名字]"
  6. 推送到远程：git push origin add-member-[你的名字]
  7. 创建 Pull Request
